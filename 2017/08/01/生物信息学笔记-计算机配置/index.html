应用软件的安装及介绍

anaconda

参考：https://zhuanlan.zhihu.com/p/25085567

sratoolkit

sratoolkit：把NCBI SRA（Sequence Read Archive）数据库中的NGS序列数据从 sra 格式转换到 fastq 格式。

fastqc

fastqc:二代测序数据质量分析软件。

看一篇：https://zhuanlan.zhihu.com/p/24608131

https://zhuanlan.zhihu.com/p/20731723

http://www.biotrainee.com/thread-1796-1-1.html

hisats

hisats：将测序结果比对到人类参考基因组上。HISAT2是TopHat2/Bowti2的继任者，使用改进的BWT算法，实现了更快的速度和更少的资源占用。

samtools

samtools:处理SAM、BAM文件的工具软件。BAM格式文件是存放高通量测序中比对结果的标准格式文件。

htseq-count

htseq-count：用于reads计数的软件，他能对位于基因组上的一些单位的reads数进行统计，这里所说的单位主要是指染色体上的一组位置区间（我们常见的就是gene exon） #### R R：统计与数据可视化工具。

软件安装

conda install -c bioconda samtools=1.5 conda install -c bioconda htseq=0.7.2 conda install -c bioconda hisat2=2.1.0 conda install -c bioconda fastqc=0.11.5 conda install -c jfear sratoolkit=2.8.1

bioconda唯一的问题就是网络问题 ,如果有国内镜像源 就非常完美了 ! 所有的软件都是可以去查询：https://bioconda.github.io/recipes.html#recipes

进阶作业，每个软件都收集一个中文教程链接，并自己阅读，发在论坛里面。 这个是优秀作业，请大家向他学习：http://www.biotrainee.com/thread-1796-1-1.html 写清楚自己是如何安装软件的，初学者体验，慢慢进步
