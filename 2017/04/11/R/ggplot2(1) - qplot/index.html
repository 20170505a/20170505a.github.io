分类变量

qplot中有各种表示分类变量的属性，例如颜色，大小等。可以将数据集中的分类变量与之相应匹配。

可以发现，color是因子类型的，可以用“颜色”属性将其区别开：

qplot(carat, price, data = dsmall, colour = color)这条命令就表示用颜色属性将数据集中的颜色分类区别开，再看一个例子，用形状将切工(cut)类型区别开：

用颜色将切工(cut)区别开：

以上的原理都一样，即一个数据集中的分类变量可以对应qplot中相应的分类属性（例如颜色是可以区分的，形状也可以是区分的，这些都能代表一种分类）。

在上面的一段命令中，as.POSIXlt()表示的是以1900年为界，将日期类型的数据转化为POSIXlt格式，as.POSIXlt(x)$year是将其中的年份减去1900后剩余的，因此，第一行要加一个1900，其中数据后面带CST，意思是标准时间。第一行构建了一个函数，这个函数的意思是提取日期类型中的年。

第二行命令，是以失业率为x轴，失业星期数的中位数为y轴绘图，并且将其中的点用路径连接起来,因此用到的参数是geom = c(“point”, “path”)。
