知识点

  • range()参数。range()这个参数的语法为range(start, end, step = 1)，range会返回一个整数序列，start为整数序列的起始值，end为整数序列的结尾值，在生成的整数序列中，不包含结尾值。例如range(0,10)就会生成0， 1， 2， 3， 4， 5， 6， 7， 8， 9这10个数字，但是0可以省略，也就是说range(0, 10)等同于range(10)。

  • random.randint(a,b)参数：返回闭区间[a, b]范围内的整数值。random.randint(0, 3)就是随机返回0, 1, 2, 3这4个数字。
